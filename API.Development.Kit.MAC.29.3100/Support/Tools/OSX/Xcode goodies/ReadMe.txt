Xcode goodies
-------------

This folder contains the following addition to the Xcode environment:
	- Archicad Add-On project template for Xcode 8+


Installation:
-------------

Place the Archicad Add-On project template (Archicad Add-On.xctemplate) into the

	/Applications/Xcode.app/Contents/Developer/Library/Xcode/Templates/Project Templates/Mac/Application Extension/

folder (create if necessary).
